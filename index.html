HTML

  • HTML stands for "HyperText Markup Language"
  • Semantic HTML Elements describe what the content and role of the element itself
  • Contents within any html element must be contained between an opening tag and an ending tag
  • While "element" and "tag" are used interchangeably, "element" refers to the entire element from the opening to the closing tag. A tag refers what's inside the angle brackets
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section element defines a section in a document that contains thematic content

CSS

  • Inline CSS utilizes styling an element directly in the HTML file
  • Internal Style Sheets contain CSS rules for the webpage in the head section
  • External Style Sheets (the most popular) allow developers to keep all their CSS rules in a separate file
  • Each CSS rule must have a selector, a property, and a value. A property plus a value is known as a declaration
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • If you're using an external style sheet, you must link it in your HTML code
  • Class Selectors allow you to give certain styling attributes to specific elements

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: makes sure the local branch is in sync with the base branch in GitHub
  • git add -A: stages all changes made in preparation for a commit
  • git commit -m "message": adds message while committing changes to branch
  • git push origin branchname: pushes changes to the remote repository

JavaScript

  • JavaScript is a programming language that allows users to interact with websites.
  • JavaScript features variables, operators, conditionals, functions, and events.
  • A variable is a named container that allows us to store data in our code.
  • Different data types can be stored in variables, such as numbers, strings, arrays, and boolean values.
  • Multiple values can be stored in a variable by using an array.
  • JavaScript can be referenced as "JS"
  • JS is case sensitive.
  • Control flow is the order in which a computer executes code in a script.
  • We can utilize conditional statements to interrupt or direct the control flow in JS. The most common conditional statement is the `if` statement.
  • Each item in an array has a unique indentifier for its location; this is called an index. Indices start with 0!
